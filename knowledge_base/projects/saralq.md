# Project: Saralq
**Type:** Developer Tooling / Browser Extension
**Tech Stack:** JavaScript, HTML/CSS, Node.js, Express

## Description
Saralq is a browser extension built for Chromium-based browsers (Chrome/Edge) that focuses on translation and proxying requests. It consists of a manifest, content scripts, and a service worker to interact directly with the browser environment.

## Backend
To support the extension during development and handle proxy requests, Saralq utilizes a custom Node.js and Express test proxy server, showcasing skills in building customized developer tooling and handling network routing.
