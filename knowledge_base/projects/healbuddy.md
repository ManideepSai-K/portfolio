# Project: Healbuddy
**Type:** ML/AI-Backed Web App
**Tech Stack:** Plain HTML/CSS/JS, FastAPI, Uvicorn, Python

## Description
Healbuddy is an application designed to provide machine learning predictions alongside a Q&A interface. It uses a lightweight, vanilla web frontend to communicate with a robust Python backend.

## AI Architecture
The backend runs on FastAPI and Uvicorn. It utilizes `scikit-learn` and `numpy` for predictive modeling. A standout feature of Healbuddy is its local LLM runtime integration using Ollama, allowing for specialized LLM modes without relying strictly on external cloud providers.
