# Project: Ingreedy
**Type:** Full-Stack AI Application
**Tech Stack:** React, Vite, FastAPI (Python), JavaScript, HTML

## Description
Ingreedy is a full-stack application that leverages machine learning and generative AI for data processing. It utilizes a React/Vite frontend for a modern, responsive user experience, backed by a high-performance FastAPI service.

## AI & Data Integration
The backend relies on `pandas`, `numpy`, and `scikit-learn` for data manipulation and traditional machine learning tasks. Crucially, it integrates the `google-generativeai` package and Google API clients to power its core LLM/GenAI features, demonstrating a strong capability in connecting cloud AI services to full-stack environments.
