# Learning As Games (LAG) Platform

## Project Description
A gamified learning platform that transforms educational content into interactive games, making learning engaging and effective.

## Architecture
- **Frontend**: React with Tailwind CSS
- **Backend**: FastAPI with PostgreSQL
- **Features**:
  - Quiz-based gameplay
  - Leaderboard system
  - Achievement badges
  - Real-time progress tracking

## Technical Highlights
- Microservices architecture
- WebSocket for real-time updates
- Redis for caching
- Docker containerization

## Results
- 1000+ active users
- 95% task completion rate
- 4.8/5 average rating
